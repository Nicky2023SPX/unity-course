Come usare Visual Studio 2022

Vedi le scritte Class1.cs e What's New?? Quelle sono le schede aperte. La prima è uno script normale di C#, mentre la seconda è una scheda che introduce le novità di VS 2022.
Esattamente sotto, nella barra con scritto Miscellaneous Files, Class1 e Class1(), si può trovare la navigazione del file. Class1 indica la classe che stai modificando, mentre Class1() indica la variabile o funzione in cui stai digitando.
Andando poi su View > Explore Solution (Visualizza > Esplora Soluzione in italiano) si possono vedere tutti i file del progetto, come se fosse una scheda Project.
