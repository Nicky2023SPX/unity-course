Come creare uno script

  1. Vai sul tuo progetto di Unity
  2. Seleziona un oggetto e premi in fondo all’Inspector “Add Component”
  3. Clicca su “New Script” e inserisci il nome dello script
  4. Clicca su Create. Clicca due volte sul nome dello script.

Vedrai del codice simile a quello seguente:

using System;
using UnityEngine;

public class YourScriptName : MonoBehaviour
{
    void Start()
    {
        
    }

    void Update()
    {
        
    }
}

Questo è il codice di base che andremo a modificare.
All’inizio del codice ci sono degli import. Il pezzo di codice “import UnityEngine;” va a dire allo script di usare la libreria base di Unity.
La parte “public class YourScriptName : MonoBehaviour” consente l’uso di comandi di Unity nello script.
La funzione “void Start()” viene chiamata soltanto all’inizio del gioco, mentre la funzione “void Update()” viene eseguita a ogni frame del gioco.
