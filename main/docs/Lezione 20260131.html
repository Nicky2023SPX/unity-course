❓Tipi, oggetti e parole chiave del C#

int numero;
float numero_decimale;
bool vero_falso;
char lettera;
enum lista_con_opzioni {a, b, c};
class classe;
void funzione() {}
public ...
private ...
static ...

Esempi pratici:

int id_livello = 2;
float posizione_x = 47.23f;
bool è_attivo = true;
char lettera = 'n';
enum stati_gioco {Idle, Lobby, Game};
class GameManager;
void apri_porte() {...}

Le parole chiave public, private e static vanno messe prima delle altre parole chiave. Ecco degli esempi:

public void lorem_ipsum() {}
static bool is_active;
private void bobo() {}
public static int number;
private static char lallala;

Come puoi vedere, si possono “impilare” queste parole chiave tra di loro.
